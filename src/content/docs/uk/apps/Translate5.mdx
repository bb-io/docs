Хмарна платформа для перекладу, перевірки, постредагування та термінології. Місія translate5 полягає у постійному розвитку системи управління перекладами з відкритим кодом та на основі хмарних технологій для мовної індустрії силами спільноти.

## Підключення 

1. Перейдіть до додатків і знайдіть Translate5. Якщо ви не можете знайти Translate5, натисніть _Add App_ у верхньому правому куті, виберіть Translate5 і додайте додаток до вашого середовища Blackbird.
2. Натисніть _Add Connection_.
3. Назвіть ваше з'єднання для майбутнього використання, наприклад, 'Мій клієнт'.
4. У полі _URL_ введіть URL вашого сервісу (він повинен мати таку структуру: https://[домен]), наприклад: https://translate5.net
5. У полі _API Key_ введіть ваш API-ключ
6. Натисніть _Connect_.
7. Підтвердіть, що з'єднання з'явилося і його статус _Connected_.

## Actions

### Comment

- **List comments**: Ця дія отримує всі коментарі для конкретного сегмента в завданні перекладу
- **Create Comment**: Створити новий коментар
- **Delete Comment**: Видалити конкретний коментар: дозволяє видалити певний коментар із сегмента

### Segment

- **List segments**: Отримує всі сегменти, пов'язані з конкретним завданням
- **Get Segment**: Отримати конкретний сегмент
- **Search Segments**: Пошук сегментів у завданні. Виконує пошук серед сегментів завдання на основі певних критеріїв, таких як текстовий вміст
- **Translate Segment**: Перекласти конкретний сегмент. Ініціює переклад вказаного сегмента.

### Task

- **List all tasks**: Отримує повний список усіх завдань у системі. Ця функціональність є важливою для користувачів, яким потрібно орієнтуватися серед різних завдань, надаючи загальний огляд поточних і завершених проєктів перекладу.
- **Get Task**: Отримати конкретне завдання: отримує детальну інформацію про певне завдання за його унікальним ідентифікатором. Ця дія дозволяє користувачам заглибитися в деталі одного завдання, надаючи відомості про його прогрес, пов'язані мови та іншу доречну інформацію.
- **Create Task**: Створити нове завдання: ініціює нове завдання перекладу з вказаними параметрами, включаючи назву завдання, мови джерела та цілі, а також інформацію про пов'язаного клієнта. Ця функція є важливою для налаштування проєктів перекладу відповідно до конкретних потреб і вимог.
- **Change Task Name**: Змінити назву завдання: оновлює назву існуючого завдання. Ця можливість коригування важлива для підтримки актуальності та ясності при розвитку проєктів з часом.
- **Delete Task**: Видалити завдання: видаляє завдання з системи. Ця функція важлива для очищення завершених або застарілих завдань, забезпечуючи організованість і фокусованість робочого простору.
- **Export Translated File**: Експортувати перекладений файл за ідентифікатором завдання: полегшує завантаження перекладеного вмісту, пов'язаного з конкретним завданням. Ця дія є ключовою для отримання остаточних перекладів, полегшуючи користувачам доступ і використання перекладених документів.
- **Create Task from ZIP**: Створити завдання з ZIP-файлу: дозволяє створювати завдання шляхом завантаження ZIP-файлу, що містить усі необхідні документи та ресурси для перекладу. Цей метод спрощує процес налаштування проєкту перекладу, забезпечуючи масове завантаження матеріалів, що робить його високоефективним для управління великими або складними завданнями.

### Translations

- **Translate Text**: Миттєвий переклад тексту з використанням вказаних мов джерела та цілі, а також обраного мовного ресурсу.

- **Translate File**: Переклад цілих файлів шляхом завантаження документа та визначення мов. Сервіс Translate5 виконує переклад і повертає перекладений документ для завантаження.

- **Write Translation Memory**: Інтеграція перекладених сегментів в OpenTM2, покращуючи багатство та корисність пам'яті перекладів.

### User

- **List all users**: Список усіх користувачів

## Webhooks

- **On task import finished**: Цей вебхук активується після завершення процесу імпорту завдання

- **On Task Finished**: Подібно до вебхуку імпорту завдання, це сповіщення активується, коли завдання досягає завершення
