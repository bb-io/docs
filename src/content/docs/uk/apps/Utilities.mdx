Цей безкоштовний додаток надає загальні утилітарні дії, які можуть бути корисними у будь-якому потоці. Він містить утиліти для роботи з датами та файлами.

## Підключення

1. Перейдіть до додатків і знайдіть Utilities. 
2. Натисніть _Add Connection_.
3. Назвіть ваше підключення для майбутнього використання, наприклад, 'Мої утиліти'.
4. Натисніть _Connect_.

## Actions

### Dates

- **Generate date** генерує дату та час на момент _виконання_. Опціонально, ви можете додати або відняти дні, години та хвилини. Це корисно для встановлення термінів у майбутньому або для генерації часових інтервалів для запитів даних. Також ви можете опціонально додати власне значення для дати. Нова згенерована дата буде відносно цієї дати.
- **Format date** приймає дату та форматує її у зручний для читання рядок. Документацію з вибору форматів можна знайти [тут](https://ss64.com/ps/syntax-dateformats.html). Також можна додати культуру (локаль).
- **Get date difference** повертає різницю між двома введеними датами в загальній кількості секунд, хвилин, годин і днів.
- **Convert text to date** конвертує текстовий ввід у дату. Опціонально можна вказати культуру (локаль).
- **Get first day of previous month** генерує дату, що відповідає першому дню попереднього місяця.
- **Get last day of previous month** генерує дату, що відповідає останньому дню попереднього місяця.

### Files

- **Get file character count** повертає кількість символів у файлі (підтримувані типи файлів: doc, docx, txt, pdf).
- **Get file word count** повертає кількість слів у файлі (підтримувані типи файлів: doc, docx, txt, pdf, html).
- **Get files word count** повертає кількість слів у файлах (підтримувані типи файлів: doc, docx, txt, pdf, html).
- **Get file name information** повертає ім'я файлу, з розширенням або без, та саме розширення.
- **Change file name** приймає файл і текстовий ввід. Ім'я файлу (без розширення) перейменовується, і файл повертається як вихідні дані.
- **Sanitize file name** видаляє вказані символи з імені файлу (без розширення).
- **Get file size** повертає розмір файлу в байтах.
- **Replace using Regex in document** працює лише з текстовими файлами (txt, html тощо). Ця дія схожа на 'Replace using Regex', але працює з файлами.
- **Extract using Regex from document** витягує текст із документа за допомогою регулярних виразів. Це працює лише з текстовими файлами (txt, html тощо) і схоже на 'Extract using Regex', але для файлів.
- **Unzip files** приймає ZIP-архів і розпаковує файли всередині нього у кілька файлів у Blackbird.

### XML Files

- **Bump version string** Збільшує рядок версії.
- **Get XML file property** Отримує властивість XML-файлу.
- **Change XML file property** Змінює властивість XML-файлу.
- **Replace XLIFF source with target** Замінює джерело XLIFF на цільове.

### Texts

- **Count characters in text** повертає кількість символів у тексті.
- **Count words in text** повертає кількість слів у тексті.
- **Sanitize text** видаляє вказані символи з тексту.
- **Extract using Regex** повертає перший збіг із наданого тексту за допомогою регулярного виразу як вхідних даних.
- **Extract many using Regex** повертає всі збіги з наданого тексту за допомогою регулярного виразу як вхідних даних.
- **Replace using Regex** використовує регулярні вирази для пошуку та заміни в тексті.
- **Convert text to document** конвертує текст у документ txt, doc або docx.
- **Convert document to text** витягує текст документа. Документ повинен бути у форматі docx/doc, pdf або txt.
- **Calculate BLEU Score** метрика, яка використовується для оцінки якості машинного перекладу тексту шляхом порівняння з еталонним текстом.
- **Split string into array** розділяє рядок на масив за допомогою вказаного роздільника.
- **Count words in texts** "Повертає кількість слів у тексті з масиву. 
  
### JSON

- **Get JSON property value** Отримує значення властивості JSON.
- **Change JSON property value** Змінює значення властивості JSON.


### Arrays

- **Array contains** перевіряє, чи містить масив певний елемент.
- **Deduplicate Array** Повертає лише унікальні елементи.
- **Array count** підраховує кількість елементів у масиві.
- **Remove entry from array** повертає масив без вказаного елемента.
- **Get first entry from array** повертає перший елемент масиву.
- **Get last entry from array** повертає останній елемент масиву.
- **Get entry by position** повертає елемент у вказаній позиції в масиві. Початкова позиція — 1.
- **Retain specified entries in array** повертає вихідний масив без елементів, яких не було в наданій контрольній групі (елементи, які потрібно зберегти).

### Numbers

- **Generate Range** генерує діапазон, надаючи початкове та кінцеве числа.
- **Convert text to number** змінює тип даних.
- **Convert text to numbers** конвертує список числових рядків у список чисел. Викидає виняток, якщо будь-яке значення не є дійсним числом.

### Scraping

- **Extract web page content** отримує необроблений і неформатований вміст з URL-адреси як текст.
- **Extract HTML content** отримує необроблений і неформатований вміст з HTML-файлу.

### Context

- **Get flight context** дозволяє отримати контекстні дані з потоку. Це включає ID та URL потоку, ID та назву птаха, а також ID та назву гнізда.

## Events
