Pangeanic створює спеціалізовані моделі штучного інтелекту, щоб ви могли створити власний генеративний ШІ (GenAI) без передачі даних третім сторонам.

## Перед налаштуванням

Перш ніж підключитися, переконайтеся, що:

- У вас є API-ключ для сервісів Pangeanic

Якщо у вас немає API-ключа, ви можете зв'язатися з [Pangeanic](https://pangeanic.com/translation-technology/translate-easy).

## Підключення 

1. Перейдіть до розділу Додатки та знайдіть додаток **Pangeanic**. Для пошуку можна використовувати пошукову функцію.
2. Натисніть _Add Connection_.
3. Назвіть ваше з'єднання для майбутнього використання, наприклад, 'Мій клієнт'.
4. У полі _API Key_ введіть ваш API-ключ.
5. У полі _URL_ введіть URL вашого API. Наприклад: http://prod.pangeamt.com:8080/
5. Натисніть _Connect_.
6. Переконайтеся, що з'єднання з'явилося і його статус _Connected_.

![PangeanicConnection](https://raw.githubusercontent.com/bb-io/Pangeanic/main/image/README/PangeanicConnection.png)

## Actions

### Text

- **Process text**: Надсилає текст для обробки та перекладу як єдиний рядок

- **Process text as array**: Надсилає текст для обробки та перекладу як масив рядків

У діях обробки тексту ви маєте можливість вказати налаштування анонімізації, включаючи чутливість, тип і теги.

### File

- **Process file**: Надсилає файл до API для обробки та перекладу, а потім отримує відповідь, що містить ID файлу

- **Download file**: Завантажує перекладений файл на основі ID файлу

## Events

### Translations

- **On file translation status updated**: Перевіряє оновлення статусу перекладу файлів. За замовчуванням перевіряє завершені переклади. Ви можете вказати статус для перевірки
