Blackbird Prompts - це колекція попередньо розроблених промптів, створених для безпроблемної роботи з будь-якою моделлю штучного інтелекту на ваш вибір.

## Actions

### Interpret
- **Interpret MQM Analysis** Інтерпретація відповіді ШІ щодо аналізу MQM

### Prompt
- **Summary prompt** Отримання промпту для узагальнення тексту
- **Generate edit prompt** Отримання промпту для редагування вхідного тексту відповідно до інструкцій
- **Post-edit MT prompt** Отримання промпту для перевірки тексту, перекладеного машинним перекладом, та створення відредагованої версії
- **Find translation issues prompt** Отримання промпту для перевірки перекладу тексту та створення коментаря з описом проблеми
- **Translate prompt** Отримання промпту для локалізації наданого тексту
- **Get localizable content from image prompt** Отримання промпту для вилучення вмісту, що підлягає локалізації, із зображення
