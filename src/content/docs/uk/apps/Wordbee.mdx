Wordbee — це потужна хмарна система управління перекладами (TMS), розроблена для підприємств, постачальників мовних послуг (LSP) та фрілансерів. Вона пропонує комплексний набір інструментів для управління всім процесом перекладу та локалізації, від створення проєкту та призначення завдань до перекладу, перевірки та доставки.

## Підключення

1.  Перейдіть до розділу Додатки та знайдіть додаток **Wordbee**. Ви можете скористатися пошуком, щоб знайти його.
2.  Натисніть _Add Connection_.
3.  Назвіть ваше підключення для майбутнього використання, наприклад, 'Моя організація'.
4. У вашому екземплярі Wordbee перейдіть до `My account` -> `API`, щоб отримати всі необхідні облікові дані API.
5.  Заповніть поле `URL` URL-адресою API вашого екземпляра.
6. Заповніть поле `Account ID` значенням властивості Account із налаштувань.
7. Заповніть поле `API key` API-ключем вашого екземпляра.
8.  Коли повернетеся до Blackbird, переконайтеся, що з'явилося підключення і його статус _Connected_.

## Actions

### Documents 

-   **Confirm delivery** підтверджує доставку певного документа.
-   **Download translated file** завантажує перекладений файл із робочого процесу.
-   **Submit new file** завантажує новий файл до існуючого проєкту.

### Orders

-   **Create order** створює нове замовлення з файлом. Необхідний формат файлу `.zip`
-   **Delete order** видаляє певне замовлення.
-   **Download order files** завантажує всі файли замовлення як ZIP-архів.
-   **Get order** повертає деталі певного замовлення.
-   **Search orders** повертає список кількох замовлень за критеріями фільтрації.

### Jobs

-   **Get job** повертає деталі певного завдання.

### Projects

-   **Get project** повертає деталі певного проєкту.
-   **Search projects** повертає список усіх проєктів у робочому просторі за критеріями фільтрації.

## Налаштування вебхуків

Після публікації птаха ви отримаєте URL зворотного виклику. Скопіюйте його, потім перейдіть до `Settings`, прокрутіть до кінця, потім `Developer API Webhooks` -> `Configure` -> `Add new webhook` і виберіть потрібний тип вебхука. У полі URL введіть URL, який ви раніше скопіювали з Blackbird. Blackbird налаштований на використання стандартних корисних навантажень, тому якщо ви щось там зміните, зверніть увагу, що вебхуки можуть не працювати належним чином. Єдине, що тут варто зазначити, це те, що ви можете обгорнути всі змінні Wordbee у корисному навантаженні лапками, щоб вебхуки стали стабільнішими. Після заповнення всіх налаштувань натисніть `Save new`.

## Events

-   **On order message** спрацьовує, коли до замовлення додається будь-яке повідомлення.
-   **On order status changed** спрацьовує, коли змінюється статус будь-якого замовлення.
-   **On project status changed** спрацьовує, коли змінюється статус будь-якого проєкту.
-   **On document workflow finished** спрацьовує, коли завершується робочий процес певного документа.
-   **On order created** спрацьовує, коли додається нове замовлення.
-   **On job status changed** спрацьовує, коли змінюється статус певного завдання.
