OneDrive — це хмарний сервіс Microsoft, який з'єднує вас з усіма вашими файлами. Він дозволяє зберігати та захищати ваші файли, ділитися ними з іншими та отримувати до них доступ звідусіль на всіх ваших пристроях.

## Підключення

1. Перейдіть до додатків і знайдіть Microsoft OneDrive.
2. Натисніть _Add Connection_.
3. Назвіть ваше підключення для подальшого використання, наприклад, 'Моя організація'.
4. Натисніть _Authorize connection_.
5. Дотримуйтесь інструкцій від Microsoft, надаючи Blackbird.io дозвіл діяти від вашого імені.
6. Коли ви повернетесь до Blackbird, переконайтеся, що з'явилося підключення і його статус _Connected_.

## Actions

### Storage

- **Upload file to folder** Завантажити файл до вказаної папки
- **Download file** Завантажити вказаний файл
- **List files in folder** Вивести метадані файлів у вказаній папці
- **List changed files** Вивести змінені файли за вказану кількість останніх годин
- **Get folder metadata** Отримати інформацію про певну папку
- **Get file metadata** Отримати інформацію про певний файл
- **Create folder in parent folder** Створити нову папку в іншій папці
- **Delete folder** Видалити вказану папку
- **Delete file** Видалити вказаний файл

> При завантаженні файлу до папки ви можете вибрати один з наступних варіантів поведінки у випадку конфлікту (коли у вказаній папці вже існує файл з такою ж назвою):
**Fail Uploading**: Якщо файл з такою ж назвою існує, завантаження зупиниться, зберігаючи оригінальний файл.
**Replace File**: Існуючий файл буде перезаписаний новим завантаженням, замінюючи його вміст.
**Rename File**: Новий файл буде збережений з унікальною назвою, зберігаючи як оригінальний, так і завантажений файли.

## Polling events

- **On files created or updated** Ця подія опитування спрацьовує, коли файли створюються або оновлюються. Ви можете вказати папку для відстеження змін та опцію `Include subfolders` для включення змін у підпапках (за замовчуванням встановлено `false`).
- **On folders created or updated** Ця подія опитування спрацьовує, коли папки створюються або оновлюються.
