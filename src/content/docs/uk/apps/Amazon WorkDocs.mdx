Amazon WorkDocs — це безпечний, повністю керований хмарний сервіс для зберігання, обміну та спільної роботи над документами. Він дозволяє користувачам створювати, редагувати та отримувати доступ до вмісту з будь-якого пристрою, забезпечуючи надійні адміністративні елементи керування для управління дозволами та відстеження активності документів.

## Actions

### Comment
- **List comments** List comments for a specific document version
- **Add comment** Add a new comment to the document
- **Delete comment** Delete specific comment

### Document
- **Get document** Get details of a specific document
- **Get document content** Get content of a specific document
- **Create document** Create a new document
- **Delete document** Delete specific document
