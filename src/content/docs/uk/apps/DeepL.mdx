DeepL — це компанія штучного інтелекту (ШІ), яка спеціалізується на послугах перекладу. Вона пропонує нейронний машинний переклад (NMT), здатний забезпечувати високоякісні переклади багатьма мовами. Система перекладу DeepL відома своєю точністю, природним звучанням перекладів та здатністю розуміти контекст.

## Перед налаштуванням

Перш ніж підключитися, переконайтеся, що:

- У вас є обліковий запис DeepL.
- У вас є API-ключ для вашого облікового запису DeepL. Його можна знайти в _налаштуваннях облікового запису_ в DeepL.
- **Важливо**: Якщо ви використовуєте API-ключ для інструментів CAT, він не працюватиме для публічного API. Натомість вам потрібно використовувати ключ автентифікації для DeepL API. Ви можете знайти [інструкції](https://youtu.be/WTt3UuiDAf4?t=79) щодо його розташування в цьому посібнику. Для отримання додаткової інформації ви також можете звернутися до цього [обговорення](https://github.com/DeepLcom/deepl-python/issues/106), де подібну проблему було вирішено.

## Підключення

1. Перейдіть до Застосунків та знайдіть додаток DeepL. Ви можете скористатися пошуком, щоб знайти його.
2. Натисніть _Add Connection_.
3. Назвіть своє з'єднання для подальшого використання, наприклад, 'Моє з'єднання DeepL'.
4. Заповніть API-ключ для вашого облікового запису DeepL.
5. Натисніть _Connect_.

## Actions

### Translation 

- **Translate text** Переклад текстових введень.
- **Translate document** Переклад документа, включаючи файли XLIFF (версії 1.2 або 2.1)

### Write
- **Improve text** покращення тексту за допомогою DeepL Write, встановлення стилю або тону написання. Наразі підтримує лише деякі мови. Також може використовуватися для зміни мовного варіанту, наприклад, з американської англійської на британську англійську. Для отримання додаткової інформації див. [документацію DeepL Write](https://developers.deepl.com/docs/api-reference/improve-text).

### Glossaries 

- **Export glossary** Експорт глосарія 
- **Import glossary** Імпорт глосарія (.tbx, .csv і .tsv) 
- **Get glossary** Отримання деталей конкретного глосарія 
- **Get glossary entries** Отримання записів глосарія у форматі TSV 
- **List glossaries** Список усіх глосаріїв
- **List glossary language pairs** Список підтримуваних мовних пар глосаріїв
- **Delete glossary** Видалення глосарія

Приклад вище показує bird, який запускається, щойно стаття публікується в Zendesk, потім ця стаття експортується як HTML-документ і перекладається через DeepL перед імпортом назад у Zendesk. При перекладі DeepL також враховує як запобіжний механізм Глосарій, який було експортовано з Microsoft Excel.
