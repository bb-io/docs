Клієнтський портал дозволяє вашим клієнтам подавати запити на проекти чи розцінки цілодобово, не чекаючи на робочі години вашого офісу. Зрештою, коли справа доходить до розцінок, швидкість - це головне. Ви можете обрати автоматичну відповідь із розцінкою або зберегти процес розцінки ручним.

## Перед налаштуванням

Перш ніж ви зможете підключитися, переконайтеся, що:

- У вас є обліковий запис XTRF.
- У вас є клієнт XTRF, від імені якого ви можете успішно увійти в клієнтський портал XTRF.
- У вас є `Username` і `Password` для клієнта XTRF.
- У вас є `Host` для клієнтського порталу XTRF. Наприклад, `organization.xtrf.eu`, це домен, який ви використовуєте для входу в клієнтський портал XTRF.

Зазвичай URL клієнтського порталу XTRF має вигляд `https://[your-xtrf-host.com]/customers`. Зауважте, що для клієнтського порталу XTRF ви використовуєте той самий хост, що й для звичайної системи XTRF; єдина відмінність - це шлях.

## Налаштування клієнта XTRF для клієнтського порталу XTRF

1. Створіть нового клієнта в XTRF або використовуйте існуючого.
2. Перейдіть на вкладку 'Main Data' та оберіть вкладку 'System Accounts'.
3. Змініть 'Access Rights', щоб встановити значення `Active`.
4. Відмітьте прапорець 'Client Portal Access Allowed'.
5. Встановіть пароль для клієнта, натиснувши кнопку 'Send Welcome E-mail' або 'Send Password Reset E-mail'. Це надішле клієнту електронний лист із кроками для встановлення пароля.
6. Ви також можете використати 'Sign in as this Partner', щоб увійти та перевірити доступ клієнта до клієнтського порталу XTRF.

## Підключення

1. Перейдіть до Apps та знайдіть додаток **XTRF Customer Portal**. Для пошуку можна використати рядок пошуку.
2. Натисніть _Add Connection_.
3. Назвіть ваше підключення для подальшого використання, наприклад, 'Підключення моєї організації'.
4. Заповніть поля `Host`, `Username` та `Password`.
5. Натисніть _Connect_.
6. Переконайтеся, що підключення успішно встановлено.

## Actions

### Invoice

- **Search invoices** - Пошук рахунків-фактур на основі заданих критеріїв.
- **Get invoice** - Отримання конкретного рахунку-фактури за його ID.
- **Download invoice** - Завантаження конкретного рахунку-фактури у форматі PDF.

### Project

- **Search projects** - Пошук проектів на основі критеріїв пошуку.
- **Get project** - Отримання проекту за його ID.
- **Create project** - Створення нового проекту.
- **Download project files** - Завантаження файлів перекладу проекту.

### Quote

- **Search quotes** - Пошук розцінок на основі заданих критеріїв.
- **Get quote** - Отримання конкретної розцінки за її ID.
- **Create quote** - Створення нової розцінки на основі наданих даних.
- **Accept quote** - Прийняття конкретної розцінки за її ID (можуть бути прийняті лише розцінки зі статусом Sent)
- **Reject quote** - Відхилення конкретної розцінки за її ID
