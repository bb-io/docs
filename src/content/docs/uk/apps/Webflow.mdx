Webflow — це хмарна платформа для веб-дизайну та розробки, яка дозволяє користувачам візуально створювати, налаштовувати та керувати адаптивними веб-сайтами. Вона надає широкий спектр функцій та інструментів для дизайнерів, розробників та підприємств, що дозволяє створювати та підтримувати професійні веб-сайти без необхідності писати код. Webflow дозволяє користувачам публікувати веб-сайти безпосередньо, співпрацювати в режимі реального часу та використовувати потужну систему керування контентом (CMS). Це робить його універсальним рішенням для окремих осіб, підприємців та компаній, які прагнуть створити та розвивати свою онлайн-присутність.

## Підключення

1.  Перейдіть до Додатків та знайдіть додаток **Webflow**. Ви можете скористатися пошуком, щоб знайти його.
2.  Натисніть _Add Connection_.
3.  Назвіть своє підключення для подальшого використання, наприклад, 'Моя організація'.
4.  Натисніть _Authorize connection_.
5.  Дотримуйтесь інструкцій, які надає Webflow.
6.  Коли ви повернетеся до Blackbird, переконайтеся, що підключення з'явилося, а його статус _Connected_.

## Actions

### Collections

-   **Get/Create/Delete collection**

### Collection items
-   **Get collection item content as HTML** повертає вміст конкретного елемента колекції у форматі HTML.
-   **Publish collection item** публікує конкретний елемент колекції. Ця дія може опублікувати елемент лише для мови за замовчуванням через обмеження API Webflow.
-   **Update collection item content from HTML** оновлює вміст конкретного елемента колекції з HTML-файлу.

### Pages

- **Search pages** Пошук сторінок за фільтрами
- **Get page content as HTML** Отримання вмісту сторінки у вигляді HTML-файлу
- **Update page content as HTML** Оновлення вмісту сторінки за допомогою HTML-файлу

	
## Events

-   **On collection item changed** запускається, коли конкретний елемент колекції змінюється
-   **On collection item created** запускається, коли створюється конкретний елемент колекції
-   **On collection item deleted** запускається, коли видаляється конкретний елемент колекції
-   **On collection item unpublished** запускається, коли конкретний елемент колекції знімається з публікації
-   **On page created** запускається, коли створюється конкретна сторінка
-   **On page deleted** запускається, коли видаляється конкретна сторінка
-   **On page metadata updated** запускається, коли оновлюються метадані конкретної сторінки
-   **On site published** запускається, коли публікується конкретний сайт
-   **On page update** запускається, коли відбувається оновлення сторінки
