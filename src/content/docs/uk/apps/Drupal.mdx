**Drupal** - це безкоштовна система керування вмістом (CMS) з відкритим кодом, яка має велику спільноту підтримки. Вона використовується мільйонами людей та організацій по всьому світу для створення та підтримки своїх веб-сайтів.

## [**TMGMT Blackbird**](https://git.drupalcode.org/project/tmgmt_blackbird) налаштування модуля

Модуль **TMGMT Blackbird** забезпечує інтеграцію з платформою Blackbird. Він дозволяє надсилати та отримувати завдання на переклад з платформи Blackbird. Більше інформації про плагін можна знайти на [сторінці плагіна Drupal TMGMT Blackbird](https://git.drupalcode.org/project/tmgmt_blackbird).

### Вимоги

Модуль залежить від [модуля Translation Management (TMGMT)](https://www.drupal.org/project/tmgmt).

### Встановлення

Встановіть модуль так само, як ви зазвичай встановлюєте сторонні модулі Drupal. Відвідайте [Встановлення сторонніх модулів](https://www.drupal.org/docs/8/extending-drupal-8/installing-contributed-modules) для отримання додаткової інформації.

### Налаштування

Перейдіть до `/admin/tmgmt/translators` та налаштуйте плагін `Blackbird`. У формі налаштування вам потрібно буде надати наступну інформацію:

- API Key: Ключ API, який використовується Blackbird для обробки завдань перекладу. Ключ API потрібно скопіювати в налаштування платформи Blackbird для завершення інтеграції.
- Щоб уникнути необхідності вручну приймати всі переклади, ви можете увімкнути налаштування `Auto accept finished translations`. Це автоматично прийматиме всі переклади після їх завершення.

### Завдання перекладу

При створенні нового завдання перекладу ви можете вибрати перекладача `Blackbird` для обробки завдання. Blackbird потім автоматично обробить завдання та поверне переклади, коли вони будуть готові. Модуль надасть наступні кінцеві точки для взаємодії `Blackbird` із завданнями перекладу:

## Перед налаштуванням

Перед підключенням переконайтеся, що:

- У вас є екземпляр Drupal
- У вас є ключ API Drupal.

## Підключення

1. Перейдіть до Додатків та знайдіть додаток **Drupal**. Ви можете скористатися пошуком, щоб знайти його.
2. Натисніть _Add Connection_.
3. Назвіть своє з'єднання для подальшого використання, наприклад, 'Моє з'єднання з Drupal'.
4. Заповніть поле `Base URL`. Наприклад: `https://your-drupal-site.com`.
5. Введіть `API key`, який ви отримали від Drupal.
6. Натисніть **Connect**.
7. Переконайтеся, що з'єднання було успішно додано.

![з'єднання](https://raw.githubusercontent.com/bb-io/Drupal/main/./image/README/connecting.png)

## Actions

- **Search jobs** - Отримати завдання за вказаними параметрами пошуку.
- **Get job as HTML** - Отримати HTML-файл із завдання з вказаним ідентифікатором завдання.
- **Update job from HTML** - Оновити завдання з HTML-файлу.

## Events

- **On translation jobs requested** - Подія на основі опитування. Повертає завдання перекладу, які були запитані після останнього часу опитування.

## Обробка помилок

У випадку помилки додаток поверне повідомлення про помилку. Повідомлення про помилку міститиме код помилки та опис помилки. Якщо ви зіткнулися з помилкою, будь ласка, зверніться до повідомлення про помилку та коду стану, щоб зрозуміти проблему. Якщо вам потрібна додаткова допомога, зверніться до нашої команди підтримки.

Приклад повідомлення про помилку: `Status code: 404, Error: Page not found | Blackbird Demo`
