Unbabel - це платформа перекладу мов, яка поєднує штучний інтелект з редагуванням людиною. Цей застосунок Unbabel здебільшого зосереджений на перекладі тексту/файлів та управлінні проєктами/файлами.

## Підключення

1. Перейдіть до застосунків і знайдіть Unbabel.
2. Натисніть _Add Connection_.
3. Назвіть своє підключення для подальшого використання, наприклад, 'Мій клієнт'.
4. Скопіюйте свої облікові дані API та вставте їх у відповідні поля у Blackbird
5. Натисніть _Connect_.
6. Переконайтеся, що підключення з'явилося і статус показує _Connected_.

## Actions

### Translations

- **(T) Translate text** надсилає простий рядок для перекладу та повертає результати, коли вони готові.
- **(T) Translate file** надсилає файл для перекладу та повертає перекладений файл, коли він готовий. Сервіс підтримує лише формати файлів txt, html та xliff.

### Projects

- **(P) Search projects** повертає список проєктів, відфільтрованих за вказаними параметрами.
- **(P) Cancel project** скасовує раніше створений проєкт, який ще не був надісланий.
- **(P) Submit project** надсилає проєкт на переклад. Після цього він не може бути оновлений або скасований.
- **(P) Create project** створює новий проєкт. Тут ви можете вказати бажані процеси обробки.
- **(P) Get project** отримує інформацію та поточний статус проєкту.
- **(P) Search project completed files** повертає список усіх завершених файлів. Кожен елемент матиме ID вхідного файлу та ID вихідного файлу. Використовуйте ID вихідного файлу разом з "Downlaod file" для завантаження перекладеного файлу.

### Files

- **(P) Upload file** завантажує новий файл до створеного профілю перед надсиланням.
- **(P) Download file** завантажує файл за його ID. Використовуйте разом з "Search project completed files" для завантаження готових файлів.

### Quality intelligence

- **(QI) Evaluate XLIFF** повертає звіт з оцінкою якості на рівні слів, речень та документа для XLIFF файлу.
- **(QI) Evaluate segment** повертає звіт з оцінкою якості на рівні слів, речень та документа для сегмента.
- **(QI) Explain XLIFF** повертає автоматичну оцінку перекладу разом з поясненням для XLIFF файлу.
- **(QI) Explain segment** повертає автоматичну оцінку перекладу разом з поясненням для сегмента.

## Events

- **(P) On project status changed** спрацьовує щоразу, коли змінюється статус проєкту. Додатково можна вказати статус, який вас цікавить, щоб, наприклад, продовжити контрольну точку лише коли проєкт доставлено.
