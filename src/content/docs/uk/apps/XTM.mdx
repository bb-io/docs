XTM Cloud - це система управління перекладами. Функції включають створення проєктів та завантаження файлів до цих проєктів для їх перекладу.

## Перед налаштуванням

Перш ніж підключитися, переконайтеся, що:

- У вас є екземпляр XTM Cloud з **увімкненим API доступом**.

## Підключення

1. Перейдіть до додатків і знайдіть XTM. Якщо ви не можете знайти XTM, натисніть _Add App_ у верхньому правому куті, виберіть XTM і додайте додаток до вашого середовища Blackbird.
2. Натисніть _Add Connection_.
3. Назвіть ваше підключення для подальшого використання, наприклад, 'My XTM'.
4. У полі клієнта введіть назву вашої компанії XTM, яка також позначена як клієнт у [документації XTM](https://api.xtm-cloud.com/rest-api/#tag/XTM-Basic/operation/generateToken).
5. Для ID користувача введіть ваш числовий ID користувача XTM. Примітка: **Це не ваше ім'я користувача**. Відомий спосіб отримати цей ID користувача - увійти в портал XTM, навести курсор на ваш аватар у верхньому правому куті та натиснути правою кнопкою миші "відкрити зображення в новій вкладці". Потім в URL цієї сторінки ви знайдете свій ID користувача.\*
6. Введіть ваш пароль користувача XTM.
7. Нарешті, введіть базову URL-адресу вашого екземпляра XTM. Наприклад, `https://xtm.mycompany.com`
8. Натисніть _Connect_.

\* Ви знаєте кращий спосіб отримати ID користувача? Повідомте нам!

## Events

>Примітка: XTM використовує URL-адреси зворотного виклику, які можна встановити лише під час створення проєкту, це означає, що лише проєкти, створені через Blackbird, працюватимуть з неопитувальними подіями. Щоб запускати Birds на основі подій щодо проєктів, які не були створені через Blackbird, використовуйте [опитувальний тригер](https://docs.blackbird.io/concepts/triggers/#polling) - ідентифікований нижче як **polling**.

- **On analysis finished**
- **On workflow transition**
- **On job finished**
- **On project created**
- **On project accepted**
- **On project finished**
- **On invoice status changed**
- **On analysis finished (polling)**
- **On projects created (polling)**
- **On projects updated (polling)**
- **On projects finished (polling)**
- **On project status changed (polling)**
