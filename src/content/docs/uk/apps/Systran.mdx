API перекладу Systran побудовано на основі RESTful API та може використовуватися в різних типах додатків.
Ця Довідка з API призначена для розробників, які хочуть створювати додатки, що взаємодіють з API перекладу SYSTRAN. 
Ви можете інтегрувати нашу технологію перекладу безпосередньо у свої внутрішні або зовнішні додатки, щоб зробити їх багатомовними або перекладати тексти й файли за допомогою API перекладу SYSTRAN.

## Перед налаштуванням

Перед підключенням потрібно переконатися, що:
- У вас є обліковий запис Systran і ви [налаштували свій обліковий запис](https://spns-dev-web.systran.us/en/authentication)
- Ви створили API-ключ у розділі `my account -> setting -> API keys`

## Підключення
1. Перейдіть до додатків і знайдіть Systran
2. Натисніть _Add Connection_.
3. Назвіть своє підключення для подальшого використання, наприклад, 'Моє підключення до Systran'.
4. Введіть свій API-ключ, отриманий раніше.
5. Введіть URL вашого екземпляра.
6. Натисніть _Connect_.


## Actions

### Corpus

- **Export corpus** Експорт файлу корпусу у форматі `.tmx` за його ID
- **Import corpus from TMX file** Додавання нового корпусу з файлу `.tmx`.

### Dictionary

- **Create dictionary** Створення словника та наповнення його за допомогою файлу `.tbx`
- **Export dictionary** Експорт словника як файл TBX
- **Update dictionary from TBX file** Оновлення існуючого словника шляхом імпорту записів з файлу `.tbx`

### File

- **Translate file** Переклад файлу з вихідної мови на цільову мову
- **Translate file (Async)** Асинхронний переклад файлу з вихідної мови на цільову мову
- **Download translated file** Завантаження перекладеного файлу за ID запиту

### Text

- **Translate text** Переклад тексту


## Events

- **On translation finished** Спрацьовує, коли статус перекладу набуває стану "завершено"
