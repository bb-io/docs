Додаток xAI дозволяє користувачам інтегрувати можливості штучного інтелекту для доповнення тексту та чату у свої робочі процеси. Незалежно від того, чи автоматизуєте ви повторювані завдання, чи генеруєте інсайти, цей додаток робить просунуті інструменти ШІ доступними.

## Перед налаштуванням

Перш ніж підключитися та почати використовувати xAI, переконайтеся у наступному:
1. У вас є активний [обліковий запис xAI](https://x.ai/) та API-ключ. Відвідайте документацію API xAI для отримання детальнішої інформації.
2. Ви авторизовані у вашому середовищі Blackbird.io.

## Підключення

1. Перейдіть до додатків і знайдіть xAI. Якщо ви не можете знайти xAI, натисніть кнопку Додати додаток у верхньому правому куті, виберіть xAI і додайте додаток до свого середовища Blackbird.
2. Натисніть Додати підключення, щоб налаштувати інтеграцію.
3. Назвіть ваше підключення для подальшого використання, наприклад, 'Моє підключення до xAI'.
4. Введіть ваш API-ключ. Ви можете згенерувати його в налаштуваннях вашого облікового запису xAI.
Формат: `xai-XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX`.
5. Натисніть Підключити, щоб завершити налаштування.

## Actions

**1. Create Completion**
Вхідні параметри:
- Model: Виберіть модель.
- Prompt: Текстовий ввід для направлення моделі.
- Max Tokens: Обмеження довжини відповіді. (Необов'язково)
- Temperature: Регулювання випадковості у відповіді (0 = детерміністичний, 1 = творчий). (Необов'язково)
- Top P: Контролює вибірку ядра. (Необов'язково)
- Stop Sequences: Вказати послідовності для зупинки завершення. (Необов'язково)
  
**Випадок використання:** Генерація творчого контенту, узагальнення тексту або відповіді на питання.

**2. Create Chat Completion**
Вхідні параметри:
- Model: Виберіть бажану модель.
- Input: Надайте повідомлення діалогу.
- Max Tokens: Вкажіть довжину відповіді. (Необов'язково)
- Temperature: Регулювання випадковості у відповіді (0 = детерміністичний, 1 = творчий). (Необов'язково)
- Top P: Контролює вибірку ядра. (Необов'язково)
- Stop Sequences: Вказати послідовності для зупинки завершення. (Необов'язково)
- Presence Penalty: Вищі значення заохочують модель досліджувати нові теми та зменшувати повторення. (Необов'язково)
- Frequency Penalty: Вищі значення знеохочують модель повторювати одні й ті самі слова або фрази кілька разів у одній відповіді. (Необов'язково)
  
**Випадок використання:** Чат-боти, віртуальні асистенти або розмовний ШІ.

Для більш детальної інформації про дії зверніться до [API reference](https://docs.x.ai/api) xAI.
