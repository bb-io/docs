Додаток Microsoft 365 Calendar, який дозволяє отримувати доступ та керувати подіями вашого календаря.

## Перед налаштуванням

Перш ніж підключитися, переконайтеся, що у вас є обліковий запис Microsoft 365.

## Підключення

1. Перейдіть до додатків і знайдіть Microsoft 365 Calendar.
2. Натисніть _Add Connection_.
3. Назвіть своє з'єднання для подальшого використання, наприклад, "Моя організація".
4. Натисніть _Authorize connection_.
5. Дотримуйтесь інструкцій від Microsoft, надаючи Blackbird.io дозвіл діяти від вашого імені.
6. Коли ви повернетеся до Blackbird, переконайтеся, що з'єднання з'явилося і його статус - _Connected_.

![Підключення](https://raw.githubusercontent.com/bb-io/Microsoft365Calendar/main/image/README/connecting.png)

## Actions

- **List calendars** повертає список календарів поточного користувача.
- **Get calendar** отримує календар. Якщо календар не вказано, повертається календар за замовчуванням.
- **Create calendar**
- **Get users' schedule information** повертає інформацію про доступність (вільно/зайнято) для групи користувачів у вказаний період часу.
- **Rename calendar** перейменовує календар. Якщо календар не вказано, перейменовується календар за замовчуванням.
- **Delete calendar**
- **List events** отримує список подій у календарі. Якщо календар не вказано, перераховуються події календаря за замовчуванням.
- **Get event**
- **List occurrences of event** повертає повторення події для вказаного діапазону часу.
- **List recently created events** отримує список подій, створених протягом останніх годин. Якщо кількість годин не вказана, перераховуються події, створені за останні 24 години. Якщо календар не вказано, перераховуються події календаря за замовчуванням.
- **List recently updated events** отримує список подій, оновлених протягом останніх годин. Якщо кількість годин не вказана, перераховуються події, оновлені за останні 24 години. Якщо календар не вказано, перераховуються події календаря за замовчуванням.
- **Create event in a calendar** створює нову подію в календарі. Якщо календар не вказано, подія створюється в календарі за замовчуванням. Якщо подія є онлайн-зустріччю, автоматично створюється зустріч Microsoft Teams. Для створення повторюваної події вкажіть шаблон повторення та інтервал, який може бути в днях, тижнях або місяцях, залежно від типу шаблону повторення. Якщо інтервал не вказано, він встановлюється на 1. Для щотижневих або щомісячних шаблонів вкажіть дні тижня, коли відбувається подія.
- **Cancel event** надсилає повідомлення про скасування та скасовує подію. Може виконуватися лише організатором зустрічі.
- **Cancel event occurrence** надсилає повідомлення про скасування та скасовує одне повторення циклічної зустрічі. Може виконуватися лише організатором зустрічі.
- **Forward event** пересилає запрошення на зустріч новому одержувачу.
- **Update event**
- **Update event occurrence**
- **Delete event**

## Events

- **On event created** спрацьовує, коли нова подія створюється у вказаному календарі.
