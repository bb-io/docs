ШІ-асистент нового покоління для ваших завдань, незалежно від їх масштабу

## Перед налаштуванням

Перш ніж підключитися, переконайтеся, що:

- У вас є [обліковий запис Anthropic](https://console.anthropic.com) та доступ до API-ключів.

## Підключення

1. Перейдіть до додатків і знайдіть Anthropic. Якщо ви не можете знайти Anthropic, натисніть _Add Connection_ у верхньому правому куті, виберіть Anthropic і додайте додаток до вашого середовища Blackbird.
2. Натисніть _Add Connection_.
3. Назвіть своє підключення для подальшого використання, наприклад, "Моє підключення до Anthropic".
4. Введіть свій API-ключ. Ви можете створити новий API-ключ у розділі [API keys](https://console.anthropic.com/account/keys). API-ключ має форму `sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`.
5. Натисніть _Connect_.

## Actions

### Chat actions

- Дія **Create completion** має такі вхідні значення для налаштування згенерованої відповіді:
1. Модель (Усі поточні та доступні моделі перераховані у випадаючому списку)
2. Запит
3. Максимальна кількість токенів для вибірки
4. Температура
5. top_p
6. top_k
7. Системний запит
8. Послідовності зупинки

Для більш детальної інформації про дію зверніться до [довідки API Anthropic](https://docs.anthropic.com/claude/docs).

### XLIFF actions

- **Process XLIFF** обробляє файл XLIFF і повертає оновлений XLIFF з перекладеним вмістом. За замовчуванням він перекладатиме вихідний текст і розміщуватиме переклад у цільовому полі. Але ви можете змінити поведінку, надавши власний `prompt`
- Дія **Post-edit XLIFF file** використовується для пост-редагування файлу XLIFF.
- Дія **Get Quality Scores for XLIFF file** використовується для отримання оцінок якості для файлу XLIFF шляхом додавання атрибута `extradata` до одиниці перекладу файлу. Критерії за замовчуванням: `fluency`, `grammar`, `terminology`, `style` та `punctuation`, але ви можете додати власні, заповнивши необов'язковий вхідний параметр `prompt`.

### Batch actions

- **(Batch) Process XLIFF file** асинхронно обробляє кожну одиницю перекладу у файлі XLIFF відповідно до наданих інструкцій (за замовчуванням просто перекладає вихідні теги) і оновлює цільовий текст для кожної одиниці.
- **(Batch) Post-edit XLIFF file** асинхронно пост-редагує цільовий текст кожної одиниці перекладу у файлі XLIFF відповідно до наданих інструкцій і оновлює цільовий текст для кожної одиниці.
- **(Batch) Get Quality Scores for XLIFF file** асинхронно отримує оцінки якості для кожної одиниці перекладу у файлі XLIFF.
- **(Batch) Get XLIFF from the batch** отримує результати пакетної обробки. Ця дія підходить лише для обробки та пост-редагування файлу XLIFF і має викликатися після завершення асинхронного процесу.
- **(Batch) Get XLIFF from the quality score batch** отримує результати оцінки якості з пакетної обробки. Ця дія підходить лише для отримання оцінок якості для файлу XLIFF і має викликатися після завершення асинхронного процесу.
