Smartcat — це хмарна платформа для перекладу та локалізації, розроблена для спрощення та автоматизації процесу перекладу. Вона поєднує систему управління перекладами (TMS), інструмент автоматизованого перекладу (CAT) та маркетплейс для перекладачів і лінгвістів.

## Actions

### Projects

- **Create project**  Створити новий проєкт
- **Delete project**  Видалити певний проєкт
- **Get project** Отримати конкретний проєкт
- **Get project statistics**  Отримати статистику для вказаного проєкту
- **List projects** Перелік усіх проєктів в обліковому записі
- **List files in a project** Перелік усіх файлів у проєкті
- **Update project**  Оновити інформацію про проєкт

### Tasks

- **Get task**  Отримати конкретне завдання проєкту
- **List project tasks**  Перелік усіх завдань проєкту

### Files

- **Download file** Завантажити конкретний файл проєкту
- **Upload file** Додати файл до файлів проєкту
- **Update file from XLIFF** Оновити вказаний файл з перекладеного XLIFF

## Events

- **On document status changed** Спрацьовує, коли змінюється статус конкретного документа
- **On project status changed** Спрацьовує, коли змінюється статус конкретного проєкту
- **On translation XLIFF import completed** Спрацьовує, коли завершується XLIFF переклад конкретного документа
- **On project created** Спрацьовує, коли створюється новий проєкт

## Translation memories
- **Import TMX file to TM** Імпортувати файл TMX у пам'ять перекладів
- **Export TMX from TM** Експортувати TMX з пам'яті перекладів
