[Custom.MT](https://custom.mt/) пропонує тонко налаштовані мовні моделі для перекладу. Це дозволяє створювати шаблони машинного перекладу та використовувати їх для автоматизованого перекладу. У Blackbird ви можете використовувати шаблони Custom.MT у своїх процесах локалізації для перекладу рядків.

## Перед налаштуванням

Перш ніж почати підключення, переконайтеся, що:

- У вас є [обліковий запис Custom.MT](https://console.custom.mt/).
- У вас є API-ключ для Custom.MT, який можна отримати [тут](https://console.custom.mt/credentials/).
- Ви створили принаймні один шаблон [тут](https://console.custom.mt/templates/).

## Підключення

1. Перейдіть до додатків і знайдіть Custom.MT. Якщо ви не можете знайти Custom.MT, натисніть _Add App_ у правому верхньому куті, виберіть Custom.MT і додайте додаток до свого середовища Blackbird.
2. Натисніть _Add Connection_.
3. Назвіть своє підключення для подальшого використання, наприклад, 'Моє підключення Custom MT'.
4. Скопіюйте свій API-ключ з консолі Custom.MT у поле API Token.
5. Натисніть _Connect_

![підключення](https://raw.githubusercontent.com/bb-io/CustomMT/main/image/README/1692783524226.png)

## Actions

- **Translate text**: Вкажіть динамічний ідентифікатор шаблону або виберіть шаблон зі списку наявних шаблонів. Додайте текст для перекладу в текстове поле. Ця дія повертає переклад тексту із використанням наданого шаблону.
- **Translate multiple texts**: Те саме, що й вище, але з кількома текстами як вхідними даними.
