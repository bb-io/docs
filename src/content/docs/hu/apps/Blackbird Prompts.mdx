A Blackbird Prompts előre megtervezett promptok gyűjteménye, amelyek zökkenőmentesen működnek bármely választott AI modellel.

## Actions

### Interpret
- **Interpret MQM Analysis** MQM elemzés AI válaszának értelmezése

### Prompt
- **Summary prompt** Prompt beszerzése szöveg összefoglalásához
- **Generate edit prompt** Prompt beszerzése a bemeneti szöveg szerkesztéséhez utasítások alapján
- **Post-edit MT prompt** Prompt beszerzése a gépi fordítás áttekintéséhez és utószerkesztett verzió generálásához
- **Find translation issues prompt** Prompt beszerzése fordítás áttekintéséhez és a problémák leírását tartalmazó megjegyzés generálásához
- **Translate prompt** Prompt beszerzése a megadott szöveg lokalizálásához
- **Get localizable content from image prompt** Prompt beszerzése képből kinyerhető lokalizálható tartalom lekéréséhez
