Az XTRF egy átfogó fordításkezelő rendszer, amely egyszerűsíti és automatizálja a fordítási munkafolyamatokat a nyelvi szolgáltatók és a vállalati fordítórészlegek számára. Robusztus projektmenedzsment, együttműködési és automatizálási funkciókat kínál a fordítási folyamatok hatékonyságának növelése érdekében.

## Actions

### Classic - Job 

- **Get job** Információk lekérése a munkáról
- **Update job** Munka frissítése, csak a frissítendő mezők megadásával
- **Upload output file for job** Kimeneti fájl feltöltése egy munkához

### Classic - Project 

- **Create language combination for project** Új nyelvkombináció létrehozása egy klasszikus projekthez feladat létrehozása nélkül
- **Create new project** Új klasszikus projekt létrehozása
- **Delete project** Klasszikus projekt törlése
- **Download file** Egy adott fájl tartalmának letöltése egy klasszikus projektben
- **Get checkbox custom field for project** Jelölőnégyzet (logikai) egyedi mező lekérése klasszikus projekthez
- **Get date custom field for project** Dátum/dátum és idő egyedi mező lekérése klasszikus projekthez
- **Get multiple selection custom field for project** Többszörös választás (lista) egyedi mező lekérése klasszikus projekthez
- **Get number custom field for project** Szám egyedi mező lekérése klasszikus projekthez
- **Get project details** Információk lekérése klasszikus projektről
- **Get text or selection custom field for project** Szöveg vagy választás egyedi mező lekérése klasszikus projekthez
- **List custom fields for project** Egyedi mezők listázása klasszikus projekthez, visszaadva azokat a mezőket, amelyek a projekthez rendelhetők, értékük nélkül. A mező értékének megszerzéséhez használja a mezőtípusnak megfelelő "Get custom field for project" műveletet
- **Update checkbox custom field for project** Jelölőnégyzet (logikai) egyedi mező frissítése klasszikus projekthez
- **Update date custom field for project** Dátum/dátum és idő egyedi mező frissítése klasszikus projekthez
- **Update multiple selection custom field for project** Többszörös választás (lista) egyedi mező frissítése klasszikus projekthez
- **Update number custom field for project** Szám egyedi mező frissítése klasszikus projekthez
- **Update project** Klasszikus projekt frissítése, csak a frissítendő mezők megadásával
- **Update text or selection custom field for project** Szöveg vagy választás egyedi mező frissítése klasszikus projekthez
- **Create payable for project** Fizetendő létrehozása klasszikus projekthez
- **Create receivable for project** Követelés létrehozása klasszikus projekthez

### Classic - Task

- **Create task for project** Új feladat létrehozása egy adott klasszikus projekthez

- **Update task in project** Projekten belüli feladat frissítése, csak a frissítendő mezők megadásával
- **Update task in quote** Árajánlaton belüli feladat frissítése, csak a frissítendő mezők megadásával
