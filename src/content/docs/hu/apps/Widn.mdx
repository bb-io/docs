A Widn egy fordítási mesterséges intelligencia, amely pontos fordításokat biztosít, megőrizve a jelentést és az árnyalatokat több mint 20 nyelvpár és különböző szakterületek között.

## Beállítás előtt

Mielőtt csatlakozhatna, győződjön meg arról, hogy:

- Rendelkezik hozzáféréssel egy Widn API kulcshoz.

## Csatlakozás

1. Navigáljon az Alkalmazásokhoz, és keresse meg a Widn alkalmazást. A keresőt használhatja a megtalálásához.
2. Kattintson az _Add Connection_ gombra.
3. Nevezze el a kapcsolatot a későbbi hivatkozáshoz, például: 'Saját Widn kapcsolat'.
4. Töltse ki az API kulcsot a Widn fiókjához.
5. Kattintson a _Connect_ gombra.

## Actions

### Translation 

- **Translate text** egyszerű szöveget fordít.

A Widn fordítási bemeneti lehetőségeket kínál a modell kiválasztásához, hangnemhez, további utasításokhoz és szójegyzékhez.

### Glossaries 

- **Import glossary** szójegyzéket importál (.tbx, .csv és .tsv).
- **Export glossary** szójegyzéket exportál
