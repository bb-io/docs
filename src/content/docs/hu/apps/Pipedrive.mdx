A Pipedrive egy ügyfélkapcsolat-kezelő (CRM) platform, amelyet az értékesítési folyamatok egyszerűsítésére és az értékesítési csapatok termelékenységének javítására terveztek. Különféle funkciókat kínál, beleértve a feladatok automatizálását, testreszabható értékesítési folyamatokat, kapcsolatkezelést és valós idejű betekintést az irányítópultokon keresztül.

## Actions

###  Activity
- **List activities** List all activities
- **List user activities** List all activities assigned to a particular user
- **Get activity** Get details of a specific activity
- **Create activity** Add a new activity
- **Update activity** Update specific activity
- **Delete activity** Delete specific activity
