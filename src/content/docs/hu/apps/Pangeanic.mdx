A Pangeanic egyedi AI modelleket készít, amelyekkel létrehozhatja saját generatív mesterséges intelligenciáját (GenAI) anélkül, hogy adatokat harmadik félnek továbbítana.

## Beállítás előtt

A csatlakozás előtt győződjön meg a következőkről:

- Rendelkeznie kell API kulccsal a Pangeanic szolgáltatásokhoz

Ha nem rendelkezik API kulccsal, vegye fel a kapcsolatot a [Pangeanic](https://pangeanic.com/translation-technology/translate-easy) céggel.

## Csatlakozás 

1. Navigáljon az Alkalmazásokhoz, és keresse meg a **Pangeanic** alkalmazást. Használhatja a keresőt is.
2. Kattintson az _Add Connection_ gombra.
3. Nevezze el a kapcsolatot a későbbi hivatkozáshoz, például: 'Saját ügyfél'.
4. Az _API Key_ mezőbe írja be az API kulcsát.
5. Az _URL_ mezőbe írja be az API URL-jét. Például: http://prod.pangeamt.com:8080/
5. Kattintson a _Connect_ gombra.
6. Ellenőrizze, hogy a kapcsolat megjelent és az állapota _Connected_.

![PangeanicConnection](https://raw.githubusercontent.com/bb-io/Pangeanic/main/image/README/PangeanicConnection.png)

## Actions

### Text

- **Process text**: Szöveget küld feldolgozásra és fordításra egyetlen szövegláncként

- **Process text as array**: Szöveget küld feldolgozásra és fordításra szövegláncok tömbjeként

A szövegfeldolgozási műveleteknél lehetősége van az anonimizálási beállítások megadására, beleértve az érzékenységet, típust és címkéket.

### File

- **Process file**: Fájlt küld az API-nak feldolgozásra és fordításra, majd lekéri a Fájl azonosítót tartalmazó választ

- **Download file**: Lefordított fájl letöltése a fájl azonosító alapján

## Events

### Translations

- **On file translation status updated**: A fájlfordítások frissítéseinek ellenőrzése. Alapértelmezés szerint a befejezett fordításokat ellenőrzi. Meghatározhatja, hogy milyen állapotot ellenőrizzen
