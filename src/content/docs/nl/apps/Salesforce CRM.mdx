Salesforce is een toonaangevend cloud-gebaseerd customer relationship management (CRM) platform ontworpen om bedrijven te helpen bij het beheren van hun klantgegevens, het verbeteren van relaties en het stimuleren van groei. Het biedt een uitgebreide reeks hulpmiddelen die verkoop, klantenservice, marketing en analyses faciliteren.

## Actions

###  Account
- **List all accounts** List all accounts
- **Get account** Get account by id
- **Create account** Create account
- **Update account field** Update account field
- **Delete account** Delete account by id

###  Contact
- **List all contacts** List all contacts
- **Get contact** Get contact by id
- **Create contact** Create contact
- **Update contact field** Update contact field
- **Delete contact** Delete contact by id

###  Files
- **List all files** List all files
- **Get file info** Get file info by id
- **Download file** Download file by id
- **Upload file** Upload file
- **Delete file** Delete file
