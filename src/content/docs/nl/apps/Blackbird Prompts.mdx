Blackbird Prompts is een verzameling vooraf ontworpen prompts die naadloos werken met elk AI-model naar keuze.

## Actions

### Interpret
- **Interpret MQM Analysis** Interpreteer AI-respons van MQM-analyse

### Prompt
- **Summary prompt** Krijg prompt voor het samenvatten van tekst
- **Generate edit prompt** Krijg prompt voor het bewerken van de invoertekst aan de hand van instructies
- **Post-edit MT prompt** Krijg prompt voor het beoordelen van MT-vertaalde tekst en het genereren van een post-edit versie
- **Find translation issues prompt** Krijg prompt voor het beoordelen van tekstvertaling en het genereren van een opmerking met de probleembeschrijving
- **Translate prompt** Krijg prompt voor het lokaliseren van de aangeleverde tekst
- **Get localizable content from image prompt** Krijg prompt voor het ophalen van lokaliseerbare inhoud uit afbeeldingen
