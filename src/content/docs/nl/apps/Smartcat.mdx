Smartcat is een cloud-gebaseerd vertaal- en lokalisatieplatform ontworpen om het vertaalproces te stroomlijnen en te automatiseren. Het combineert een vertaalmanagementsysteem (TMS), een computerondersteund vertaalprogramma (CAT) en een marktplaats voor vertalers en taalkundigen.

## Actions

### Projects

- **Create project**  Een nieuw project aanmaken
- **Delete project**  Een specifiek project verwijderen
- **Get project** Een specifiek project ophalen
- **Get project statistics**  Statistieken voor het opgegeven project ophalen
- **List projects** Alle projecten in het account weergeven
- **List files in a project** Alle bestanden in een project weergeven
- **Update project**  Projectinformatie bijwerken

### Tasks

- **Get task**  Een specifieke projecttaak ophalen
- **List project tasks**  Alle projecttaken weergeven

### Files

- **Download file** Een specifiek projectbestand downloaden
- **Upload file** Bestand toevoegen aan projectbestanden
- **Update file from XLIFF** Het opgegeven bestand bijwerken vanuit een vertaalde XLIFF

## Events

- **On document status changed** Wordt geactiveerd wanneer de status van het specifieke document is gewijzigd
- **On project status changed** Wordt geactiveerd wanneer de status van het specifieke project is gewijzigd
- **On translation XLIFF import completed** Wordt geactiveerd wanneer de XLIFF-vertaling van het specifieke document is voltooid
- **On project created** Wordt geactiveerd wanneer een nieuw project is aangemaakt

## Translation memories
- **Import TMX file to TM** Een TMX-bestand importeren in translation memory
- **Export TMX from TM** Een TMX exporteren uit translation memory
