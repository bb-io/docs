Asana is een platform voor projectmanagement en teamcollaboratie. Het biedt functies zoals taakbeheer, teamcommunicatie, projectplanningtools en een Tijdlijnweergave. Deze Asana-applicatie richt zich voornamelijk op project- en taakbeheer.

## Verbinden

1.  Ga naar apps en zoek naar Asana. Als je Asana niet kunt vinden, klik dan rechtsboven op _Add App_, selecteer Asana en voeg de app toe aan je Blackbird-omgeving.
2.  Klik op _Add Connection_.
3.  Geef je verbinding een naam voor toekomstig gebruik, bijvoorbeeld 'Mijn klant'.
4.  Ga naar de Asana-ontwikkelaarsconsole (https://app.asana.com/0/my-apps) en klik op _Create token_.
5.  Voer de naam in, ga akkoord met de voorwaarden van Asana
6.  Kopieer de API-token en plak deze in het juiste veld in Blackbird
7.  Klik op _Connect_.
8.  Controleer of de verbinding is verschenen en de status _Connected_ is.

## Actions

### Attachments

- **List attachments** geeft alle bijlagen van het gespecificeerde object weer. Object ID kan staan voor een ID van een `Project`, `Project brief` of `Task`.
- **Get/upload/delete attachment**.

### Custom fields

- **Get text/date/enum custom field** geeft de waarde van een aangepast veld met tekst/datum/enum-type voor een specifieke taak.
- **Update text/date/enum custom field** stelt de waarde in van een aangepast veld met tekst/datum/enum-type voor een specifieke taak.

### Projects

- **List projects** geeft alle projecten weer. Je kunt ook `Workspace` of `Team` specificeren om projecten van een specifieke werkruimte of team op te halen. Ook kun je de variabele `Archived` gebruiken om alleen gearchiveerde projecten op te halen of vice versa.
- **Get/create/update/delete project**.
- **Get project sections** geeft alle secties in het gespecificeerde project weer.
- **Get project status** geeft een volledig record voor een enkele statusupdate
- **Get project status updates** geeft statusupdate-records voor alle updates op het project

### Sections

- **List sections** geeft alle secties van het gespecificeerde project weer.
- **Get/create/update/delete section**.

### Tags

- **List tags** geeft alle tags weer. Je kunt ook `Workspace` specificeren om tags van een specifieke werkruimte op te halen.
- **Get/create/update/delete tag**.

### Tasks

- **List tasks** geeft alle taken weer van de gespecificeerde `Workspace`, `Project`, `Tag`, `Section` of `User task list`.
- **Get/create/update/delete task**.
- **Get tasks from a user task list** geeft een lijst van taken in de Mijn Taken-lijst van een gebruiker.
- **Get tasks by tag** geeft alle taken met de opgegeven tag weer.
- **Assign tag to task** voegt een tag toe aan een taak.

### Users

- **List users** geeft alle gebruikers weer. Je kunt ook `Workspace` of `Team` specificeren om gebruikers van een specifieke werkruimte of team op te halen.
- **Get user** geeft details van een gespecificeerde gebruiker.
- **Get user's task list** geeft een volledig record voor de takenlijst van een gebruiker.
- **Get user's teams** geeft alle teams weer waaraan de opgegeven gebruiker is toegewezen.

### Workspaces

- **List workspaces** geeft alle werkruimtes weer die zichtbaar zijn voor de geautoriseerde gebruiker.
- **Get workspace** geeft details van een gespecificeerde werkruimte

## Events

### Projects 
- **On projects added** wordt geactiveerd wanneer projecten worden toegevoegd.
- **On projects changed** wordt geactiveerd wanneer projecten worden gewijzigd.
- **On projects deleted** wordt geactiveerd wanneer projecten worden verwijderd.
- **On projects removed** wordt geactiveerd wanneer projecten worden verwijderd.
- **On projects undeleted** wordt geactiveerd wanneer projecten worden hersteld.

### Tasks

- **On tasks changed** wordt geactiveerd wanneer taken worden gewijzigd.
- **On tasks deleted** wordt geactiveerd wanneer taken worden verwijderd.

- **On tasks undeleted** wordt geactiveerd wanneer taken worden hersteld.
