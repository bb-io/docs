Amazon Textract is een machine learning-service die automatisch tekst, handschrift en gegevens extraheert uit gescande documenten, inclusief formulieren en tabellen. Het gaat verder dan eenvoudige optische tekenherkenning (OCR) door de structuur van documenten te identificeren, waardoor het mogelijk wordt om complexe gegevensvelden en hun onderlinge relaties te extraheren, wat nuttig is voor het automatiseren van documentverwerking en gegevensinvoer.

## Actions

### Analysis
- **Analyze document** Analyze an input document for relationships between detected items
- **Analyze expense** Analyze an input document for financially related relationships between text
- **Analyze identity document** Analyze identity document for relevant information
- **Detect document text** Detect text in the input document
