## Voordat je begint

- Je moet een account hebben op FlowFit.
- Je hebt een login en wachtwoord nodig voor je account op FlowFit.

## Verbinding maken

1. Navigeer naar Apps en zoek de **FlowFit** app. Je kunt de zoekfunctie gebruiken om deze te vinden.
2. Klik op _Add Connection_.
3. Geef je verbinding een naam voor toekomstig gebruik, bijvoorbeeld 'Mijn verbinding'.
6. Vul de login voor je FlowFit-account in.
7. Vul het wachtwoord voor je FlowFit-account in.
8. Klik op _Connect_.
9. Bevestig dat de verbinding is verschenen en dat de status _Connected_ is.

## Actions

### Projects

- **Get project** haal informatie op over een project.
- **Create project** maakt een nieuw project aan.
- **Delete project** verwijdert het opgegeven project, samen met alle bijbehorende entiteiten.

### Project documents

- **Get project document** haal informatie op over een projectdocument.
- **Download source documents** download brondocumenten voor een project.
- **Download reference documents** download referentiedocumenten voor een project.
- **Download deliverable documents** download op te leveren documenten voor een project.
- **Download deliverable document** download een op te leveren projectdocument gespecificeerd door de project- en taak-ID's.
- **Upload source document** upload een brondocument naar het opgegeven project.

### Tasks

- **Get source document** Haal informatie op over het opgegeven brondocument van het project en download het.

## Events

- **On project documents delivered** geeft projectdocumenten terug die zijn opgeleverd na de laatste polling-tijd (op polling gebaseerde gebeurtenis).
- **On projects status changed** geeft projecten terug waarvan de status is gewijzigd na de laatste polling-tijd (op polling gebaseerde gebeurtenis).
