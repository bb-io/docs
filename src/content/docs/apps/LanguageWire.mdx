LanguageWire is a comprehensive language service platform tailored for translation and localization needs. Offering a suite of powerful features, LanguageWire streamlines project management, facilitates seamless collaboration, and enhances efficiency throughout the translation process. In essence, LanguageWire app provides a robust platform that automates labor-intensive tasks, enhances collaboration, and streamlines translation workflows, empowering users to achieve greater efficiency and productivity in their language service endeavors. This LanguageWire app allows you to automate many of the labor-intensive tasks.

## Before setting up

Before you can connect you need to make sure that:

- You must [be registered](https://developers.languagewire.com/project-api/get-started/prerequisites#01-platform-registration) as a LanguageWire user
- Log in to our Platform via the [Dashboard](https://developers.languagewire.com/dashboard/)
- Create your [live applications](https://developers.languagewire.com/project-api/guides/apps-management/#create-your-application)
- Generate your [apps credentials](https://developers.languagewire.com/project-api/guides/authentication/#generate-credentials)

## Connecting

1. Navigate to apps and search for LanguageWire
2. Click _Add Connection_.
3. Name your connection for future reference e.g. 'My LanguageWire connection'.
4. Fill 'Client ID' field with your client id (you can get it from [dashboard](https://developers.languagewire.com/dashboard/))
5. Fill 'Client Secret' field with your client secret (you can also get it from [dashboard](https://developers.languagewire.com/dashboard/)). Note, that every time you get this value it generate new key and previous one become invalid
7. Click _Connect_

## Actions

### Projects

- **Get projects:** Retrieves a list of projects based on applied filters
- **Get project:** Fetches details for a specific project by ID
- **Create project:** Creates a new project with provided details and settings
- **Delete project:** Deletes a specified project by ID
- **Send project for translation:** Initiates the translation process for a project, with an option to send as a quote

### Files

- **Download file:** Downloads a specific file
- **Upload file:** Uploads a new file, with support for specifying source and target languages
- **Upload reference file:** Uploads a new reference file for a specific project
- **Get file metadata:** Retrieves metadata for a specified file
- **Delete file:** Deletes a specified file, with differentiation between project and reference files
- **Download all project files:** Retrieves all files associated with a project
